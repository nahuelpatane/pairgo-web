## Brand at a glance

**Primary color:** `#E8654A` — Coral  
**Display font:** Fraunces (serif) — headings, logo  
**Body font:** DM Sans (sans-serif) — UI, copy  
**Background:** `#FAFAF7` — warm off-white

### Logo rules
- Use `logo-lockup.svg` (icon + wordmark) wherever space allows
- Use wordmark alone only when the icon appears nearby in context
- The "go" in *pair**go*** is always coral (`#E8654A`) — never change this
- Minimum wordmark width: **60px** — below that, use icon only
- Maintain clearspace of 1× cap-height on all sides

### Don'ts
- Don't recolor or change the coral accent on any logo variant
- Don't place the dark wordmark on dark backgrounds (use the light variant)
- Don't stretch or distort the logo
- Don't add drop shadows to the SVG logos
